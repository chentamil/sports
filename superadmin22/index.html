✅ Attendance

Attendance is saved per batch. If a student is in 2 batches (e.g. Morning + Evening), pick the date, select Morning → mark → Save All, then select Evening for the same date → mark → Save All again. Already-marked days are editable — just reload that date/batch, change the dropdown, and Save All to overwrite.

Mark a Holiday

This marks every active student in every active batch as "Holiday" for the selected date — useful for tournaments, festivals, or academy closures. Any existing attendance for that date will be overwritten.
